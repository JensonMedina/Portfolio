Resumen

¡Hola! Soy Jenson, un apasionado desarrollador con experiencia en el mundo de la programación. Comencé mi viaje con la programación a través del curso gratuito de Maxi Programa, donde adquirí mis primeras habilidades y desarrollé una sólida base lógica.

Contaba con un background en Economía, lo que me proporcionó una ventaja, ya que estaba familiarizado con conceptos matemáticos y estadísticos, facilitando mi comprensión de la lógica de programación. Posteriormente, avancé a través de los niveles 1 y 2, donde adquirí conocimientos fundamentales de C# y trabajé con tecnologías como .NET Framework, WinForms y SQL Server. Al descubrir la Tecnicatura en Programación en la UTN, decidí inscribirme en la carrera.

En la universidad, amplié mis habilidades trabajando con diferentes lenguajes como C, C++, y Python, explorando conceptos avanzados como programación orientada a objetos, polimorfismo y modularidad. En el laboratorio, me sumergí en el desarrollo web con HTML, CSS, JS y Bootstrap. Actualmente, estoy cursando el nivel 3 de Maxi, centrándome en tecnologías web como ASP.NET y WebForms. Además, he profundizado en HTML, CSS y JavaScript por cuenta propia, complementando mi formación académica.

Proyectos

  • BullCoin

    Tecnologías: ASP.NET - C# - CSS - JavaScript - SQL Server
    Haz clic aquí

    BullCoin es un cotizador de divisas, se pueden consultar las cotizaciones de distintos tipos de monedas, por ejemplo el dólar, euro, entre otros. Las cotizaciones se pueden guardar en favoritos y visualizar en la página de mis archivos. Luego, en la página de informes, se realiza un informe con las cotizaciones guardadas en favoritos, en donde se indica la variación de las divisas en distintas fechas.

  • Ferretería "De eso no tengo"

    Tecnologías: C# - WinForms - SQL Server
    Haz clic aquí

    Ferretería "De eso no tengo" es una aplicación de escritorio de gestión de artículos para una ferretería. La aplicación te permite dar de alta artículos, modificarlos o eliminarlos. Se pueden buscar artículos con un filtro de búsqueda, también hay una opción para poder cambiar el precio de varios artículos al mismo tiempo por el criterio de marca.

  • ARGENNTUM

    Tecnologías: HTML - CSS - Bootstrap - JavaScript
    Haz clic aquí

    Argenntum es el sitio web de un banco ficticio. Se cuenta su historia, se ofrecen distintos servicios, se vende como el mejor banco y te invita constantemente a que te unas como cliente. Este proyecto fue desarrollado en grupo para el curso de FullStack Python de Codo a Codo.
